This is a plugin wrapper for KekuleHtml for Ahnenblatt:
https://www.ahnenblatt.de/

It needs to be placed into:
%USERPROFILE%\Documents\Ahnenblatt\PlugIns

like this:
Documents/
├─ Ahnenblatt/
│  ├─ PlugIns/
│  │  ├─ KekuleHtml/
│  │  │  ├─ ...
│  │  │  ├─ KekuleHtmlUi.exe
│  │  │  ├─ [and everything else from the published archive]
│  │  ├─ KekuleHtml.abp

If everything was put into the right place KekuleHtml will show up in Ahnenblatt in the menu under Tools/KekuleHtml/Start.
